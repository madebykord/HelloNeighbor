# Steam версии Hello Neighbor

## Важная пометка
**Вы должны владеть [Hello Neighbor в Steam](https://store.steampowered.com/app/521890/Hello_Neighbor/)** чтобы версии работали (кроме Hello Guest Prototype).
Это будет работать только на **ПК с Windows**.

## Туториал

 1. **Откройте архив используя 7-Zip или WinRAR**
 2. **Откройте папку "Hello Neighbor"** и **нажмите ПКМ по папке версии** в которую вы хотите играть
 3. **Нажмите *Копировать В...*** и **выберите путь** в котором будет находится папка с версией
 4. **Откройте папку которую вы скопировали**
 5. **Откройте файл "HelloNeighbor.exe"** если вы выбрали любую из версий Hello Neighbor
 6. **Нажмите ПКМ по файлу "Launcher.exe"** и **select Run as administrator** если вы выбрали Hello Guest Prototype

**Выпущена 9 Декабря, 2017**

### Hello Neighbor (v1.1.2)
**Выпущена 11 Декабря, 2017**

Добавлен забор напротив красного окна в 1-ом Акте
Были добавлены Детали Дружелюбного Режима
Изменена модель световой панели
Правая сторона жёлтой стены рядом с решёткой (где находится трамплин во 2-ом Акте) недосягаема.
Добавлен интерьер в пустые комнаты 3-его Акта
Добавлены перегородки в затопленную комнату во 2-ом Акте.
Анимация прыжка соседа теперь имеет анимацию рук вместе с телом.

Изменены локации манекенов

Больше комнат имеют декорации

Некоторые предметы перемещены в другие локации
